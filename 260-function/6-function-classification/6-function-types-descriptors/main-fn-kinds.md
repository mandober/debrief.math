# Main function kinds

- Constant function, `b`
- Linear function, `ax + b`
- Quadratic function, `ax² + bx + c`
- Cubic function, `ax³ + bx² + cx + d`
- Quintic function, `ax⁴ + bx³ + cx² + dx + e`
- Polynomials, `aₙxⁿ`
- Power, `axᵖ`, `xᵃˣᐩᵇ`
- Rational, `P(x)/Q(x)`
- Trig functions, sin(x)
- Logarithmic, `ln(ax+b)`
- Exponential functions, `aᵃˣᐩᵇ`
- Exponential function, `eᵃˣᐩᵇ`
