# Epimorphism

https://ncatlab.org/nlab/show/epimorphism



```
                                g₁
           f         -------------->
a ● --------------> ● b             ● c
                     -------------->
                                g₂


e is an epimorphism iff
  ∀ f g ∈ hom(b,x).
    ∃ e ∈ hom(a,b).
      (f ∘ e = g ∘ e) -> f = g
```
