# Functions in CS

* Functions in PL
  - main-effect
  - side-effect
  - honest
  - pure
  - diverging function
  - computable function
  - applicative functor
  - functor
  - closure





* Function applications
    - function application
      - regular application: using a fn to map a value, `f x`
    - Functor
      - fmap: using a fn to map a structure, `f [x]`
    - Applicative functor:
      - ap: using a fn in a context to map a structure
    - Monad:
      - bind: using a fn in a context to map a structure
        returning a nested structure

* Function time complexity classes
  - polynomial time
  - constant,     O(1)
  - linear,       O(n)
  - logarithmic,  O(log n)
  - linearithmic, O(n log n)
  - quadratic,    O(n²)
  - cubic,        O(n³)
  -               O(2ⁿ)
  -               O(3ⁿ)
  -               O(kⁿ)
