# Rules of inference

## Inference rules

```js
-------------------------------------------------------------------------------
:Conjunction:
===============================================================================

A true   B true                      A ⋀ B true        A ⋀ B true
--------------- ∧I                  ------------ ∧Eʟ  ------------ ∧Eʀ
   A ⋀ B true                          A true            B true



-------------------------------------------------------------------------------
:Disjunction:
===============================================================================

                                                  [A]ⁱ true  [B]ʲ true
                                                   ⫶           ⫶
  A true           B true             A ∨ B true   C true     C true
---------- ∨I₁   ---------- ∨I₂       ---------------------------------- ∨Eᵢⱼ
A ∨ B true       A ∨ B true                       C true



-------------------------------------------------------------------------------
:Implication:
===============================================================================

 [A]¹ true
  ⫶
  B true            A true   A → B true
----------- →I¹    --------------------- MP
A → B true                B true



-------------------------------------------------------------------------------
:Negation:
===============================================================================

 [A]¹ true
  ⫶
  p true            A true    ¬A true
----------- ¬I¹     ------------------ ¬E
 ¬A true                  C true



-------------------------------------------------------------------------------
:Truth:
===============================================================================

-------- ⟙I
⟙ true



-------------------------------------------------------------------------------
:Falsum:
===============================================================================

⟘ true
------- ⟘E
C true



-------------------------------------------------------------------------------
:Universal:
===============================================================================

[a/x]A true               ∀x. A true
------------ ∀Iᵃ          ------------ ∀E
  ∀x.A true               [t/x]A true



-------------------------------------------------------------------------------
:Existential:
===============================================================================

                                       [[a/x]A true]
                                             ⫶
[t/x]A true              ∃x.A true        C true
------------ ∃I          ----------------------------- ∃Eᵃ
  ∃x.A true                        C true


```

```js
          C

```

+ negation `¬p := p -> ⟘`
+ equivalence `p <=> q := (p -> q) ⋀ (q -> p)`





## Set 3

```js
// CONJUNCTION

A true   B true       A ∧ B true       A ∧ B true
--------------- ∧I    ---------- ∧E₁   ---------- ∧E₂
   A ∧ B true           A true           B true

A   B       A ∧ B       A ∧ B
----- ∧I    ----- ∧E₁   ----- ∧E₂
A ∧ B       A               B

// DISJUNCTION
                                               [A]ⁱ true  [B]ʲ true
                                                ⫶           ⫶
  A true           B true          A ∨ B true   C true     C true
---------- ∨I₁   ---------- ∨I₂    ---------------------------------- ∨Eᵢⱼ
A ∨ B true       A ∨ B true                    C true

// IMPLICATION

 [A]¹ true
  ⫶
  B true            A true   A → B true
----------- →I¹    --------------------- MP
A → B true                B true

// NEGATION

 [A]¹ true
  ⫶
  p true            A true    ¬A true
----------- ¬I¹     ------------------ ¬E
 ¬A true                  C true

// TRUTH                      FALSE

                             ⟘ true
-------- ⟙I                 ------- ⟘E
⟙ true                       C true



// UNIVERSAL QUANTIFICATION

[a/x]A true               ∀x. A true
------------ ∀Iᵃ          ------------ ∀E
  ∀x.A true               [t/x]A true


// EXISTENTIAL QUANTIFICATION
                                       [[a/x]A true]
                                             ⫶
[t/x]A true              ∃x.A true        C true
------------ ∃I          ----------------------------- ∃Eᵃ
  ∃x.A true                        C true
```
