# sin

- dom(sin) = ℝ
- ran(sin) = [-1,1]
- intercepts
  - x-intercept: every `πn` (n ∈ ℤ)
  - y-intercept: one intercept at (0,0)
- parity: odd
- extrema
  - minima: (2πn + 3π/2, 1)
  - maxima: (2πn + π/2, 1)
  - inflection points: every (πn, 0)
- monotonicity
  - monotonically decreasing: (2πn +  π/2, 2πn + 3π/2)
  - monotonically increasing: (2πn + 3π/2, 2πn + 5π/2)
- period: 2π
- (in above, `n ∈ ℤ`)



>sin²x = (sin x)² = sin²(x) = (sin(x))² ≠ sin(x²)
