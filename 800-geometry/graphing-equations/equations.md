# Equations

Line
- slope-intercept line: y = mx + b
- point-point line: y - y₀ = m(x - x₀)

Circle
- unit circle: x² + y² = 1
- north semicicle: y = √(x² - 1)
- south semicicle: y = -√(x² - 1)
- circle of radius r centered at (a,b): (x/a)² + (y/b)² = r²

Parabola
- y = x² + C

Hyperbola

Ellipse
- x² - y² = r²
- (x/a)² - (y/b)² = r²

Trig
- sin x
- cos x
- tan x = sin x / cos x
- sec x
- cot x
- csc x

Polynomials
- degree 0
  - constant, c
- degree 1
  - x + c = d
- degree 2, quadratic
  - 2x² − 3x + 5 = 0
- degree 3, cubic
  - x³ − 2x² − x - 1 = 0

Elliptical curves
- x³ + x² = y² + y

Logarithmic
- logₐ(x)
- ln(x)
- 1/ln(x)

Others
- reciprocal
  - 1/x, 1/x²
  - 1/ln x
  - 1/x
- inverse of function
- function combination (+, -, ×, ÷)
- function composition (f ∘ g)
