# Math Primer: Logic connectives

Logic connectives
- $$\lnot $$ negation
- $$\land $$ conjunction
- $$\lor  $$ disjunction
- $$\to   $$ implication
- $$\equiv$$ iff

Logical constants
- $$\top  $$ true
- $$\bot  $$ false


Negation, `¬`
- `p ∨ ¬p`
- *"p or not p"*
- that is, a statement is either true or not true



Conjunction `∧`
- Disjunction `∨`
- Implication `→`
- IFF         `↔`

* `(p = T) → (¬p = F) ∧ (¬¬p = T)`
  - "`IF` p is TRUE, `THEN` NOT p is FALSE `AND` NOT NOT p is TRUE"
* `p → ¬¬p = p`
  - "if p, then not not p, is p"
  - "if p is T, then not not p is T"
