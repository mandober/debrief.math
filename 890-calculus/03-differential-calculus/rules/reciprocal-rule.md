# Reciprocal rule

The reciprocal rule can be derived from the quotient rule.

```js
⎛  1  ⎞′   -g′
⎜---- ⎟  = ----
⎝  g  ⎠     g²

⎛   1  ⎞′  -g′(x)
⎜------⎟ = ------
⎝ g(x) ⎠    g²(x)

// Deriving the reciprocal rule from the quotient rule
⎛  f  ⎞′  f′g - fg′
⎜---- ⎟ = --------- // the quotient rule
⎝  g  ⎠      g²

⎛  1  ⎞′  1′g - 1g′    0g - g′    0 - g′    -g′
⎜---- ⎟ = --------- = -------- = ------- = ----
⎝  g  ⎠      g²          g²        g²       g²
```
