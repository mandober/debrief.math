# Unit type

The unit type is the canonical type inhabited with a single term.
- the type is denoted by: `⊤`, `𝟙`
- the term is denoted by: `tt`, `⋆`

e.g. `⊤ : ⋆` or `⊤ : tt` or `𝟙 : tt`

The unit type can actually be considered as an empty product (empty pair) type.

Rules
- one intro rules
- no elim rule
