https://en.wikipedia.org/wiki/Time_complexity

* Function time complexity classes
  - polynomial time
  - constant,     O(1)
  - linear,       O(n)
  - logarithmic,  O(log n)
  - linearithmic, O(n log n)
  - quadratic,    O(n²)
  - cubic,        O(n³)
  -               O(2ⁿ)
  -               O(3ⁿ)
  -               O(kⁿ)
