# Set constructions :: Von Neumann construction of natural numbers


- S(n) := n ⋃ {n}
- 0 := ∅
- 1 := {∅}
- 2 := {∅, {∅}}
- 3 := {∅, {∅}, {∅,{∅}}}
- 4 := {∅, {∅}, {∅,{∅}}, {∅,{∅},{∅,{∅}}}}
- 5 := {∅, {∅}, {∅,{∅}}, {∅,{∅},{∅,{∅}}}, {∅,{∅},{∅,{∅}},{∅,{∅},{∅,{∅}}}}}
- n := {0, 1, 2, …, n - 1}
- S(n) = { 0, 1, 2, …, n }

Properties
- ω = ℕ
- m < n ⇒ m ⊆ n
- ω is a transitive set
- n ∈ ω ⇒ n is transitive set
- n ∈ ω ⇒ n is swollen set
