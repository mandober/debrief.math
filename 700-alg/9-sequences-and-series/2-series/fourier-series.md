# Fourier series

Mathematical way to represent *non-trig periodic functions* as *infinite sums of trig functions*.

*Square wave* function is an example of a non-trig periodic function.

```
Square wave function

          ↑ᶠ⁽ᵗ⁾
          │
  ┌───┐   ├───┐    ┌───┐
  │   │   │   │    │   │
──┴───┴───┼───┴────┴───┴────→ᵗ
 ⁻²  ⁻¹  ⁰│   ¹    ²   ³
```
